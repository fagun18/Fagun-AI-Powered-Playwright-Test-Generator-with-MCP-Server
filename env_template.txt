# API Keys Template
# Get your API keys from the respective providers

# Google Gemini API Key
# Get your API key from: https://makersuite.google.com/app/apikey
GEMINI_API_KEY=your_gemini_api_key_here

# Grok API Key (X.AI)
# Get your API key from: https://console.x.ai/
GROK_API_KEY=your_grok_api_key_here

# Instructions:
# 1. Copy this file to .env
# 2. Replace 'your_gemini_api_key_here' with your actual Google Gemini API key
# 3. Replace 'your_grok_api_key_here' with your actual Grok API key
# 4. The system will automatically use Gemini first, then fallback to Grok if quota is exceeded
# 5. Never commit the .env file with real API keys to version control
# 6. Keep your .env file secure and don't share it publicly
